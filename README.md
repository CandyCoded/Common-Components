# Common Components

## Features

-   BillboardTowardsCamera
-   Gizmo
-   Logger
-   RigidbodyMonoBehaviour
-   ScreenShake
-   SelfDestructParticleSystem

## Installation

### Unity Package Manager _(Unity 2018.3)_

<https://docs.unity3d.com/Packages/com.unity.package-manager-ui@2.0/manual/index.html>

```json
{
    "dependencies": {
        "com.candycoded.common-components": "https://github.com/CandyCoded/Common-Components.git#upm"
    }
}
```

### Git Submodule

#### Install

```bash
$ git submodule add git@github.com:CandyCoded/Common-Components.git Assets/Plugins/Common-Components/
```

After cloning repo with submodule already installed

```bash
$ git submodule init && git submodule update --remote
```

#### Update

```bash
$ git submodule update --remote
```
